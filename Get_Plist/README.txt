Get Person List v1.2
(extract_globalid_source)

소개
 - 폴더 내 액셀파일(.xls)의 첫 시트의 원하는 행을 .xlsx로 추출하는 프로그램

상세정보
 - 폴더 내 .xls파일들의 행을 추출
	-  출력하는 행 : 1, 23, 24, 25, 27
		- region_id, 성, 명, 출생년도, 간지
 - 출력하는 행의 정보를 [해당 년도_plist.xlsx]파일로 출력

 사용법
  1. 추출하고자 하는 파일이 있는 위치로 프로그램을 복사
  2. 프로그램을 더블 클릭
  3. 프로그램을 실행한 위치에 output 디렉토레 출력물 저장
